# Programação 4 - HTML, CSS e JavaScript

Bem-vindo ao repositório da disciplina **Programação 4**! Este espaço foi criado para organizar e entregar as atividades práticas que compõem as **notas parciais** da matéria. Aqui você encontrará exercícios, projetos e exemplos que envolvem as três tecnologias fundamentais para o desenvolvimento web front-end.

## 📚 Sobre a disciplina

A ementa de Programação 4 tem como objetivo consolidar os conhecimentos de construção de interfaces web e interatividade, utilizando:

- **HTML** – estruturação semântica do conteúdo
- **CSS** – estilização, layout responsivo e design
- **JavaScript** – programação de comportamento, manipulação do DOM, eventos e lógica assíncrona

Ao longo do curso, os alunos desenvolverão pequenos projetos incrementais que simulam situações reais de desenvolvimento web.

## 🎯 Finalidade deste repositório

Este repositório serve como portfólio de entregas para avaliação parcial. Cada trabalho ou atividade estará devidamente organizado em pastas com seus respectivos arquivos `.html`, `.css` e `.js`. Os commits e versões registradas aqui serão considerados para fins de correção e atribuição de nota.

## 🧠 Como estudar / usar este conteúdo

Sinta-se à vontade para:

- Clonar o repositório e executar os exemplos localmente
- Usar o código como base para seus próprios estudos
- Adaptar as soluções para outros contextos, respeitando os termos da licença

A estrutura básica do projeto segue o padrão:

│ ├── style.css

└── ...
